| --- | --- | --- | --- |
| id | 主键 | id |  |
| chunk | 文档片段 | text |  |
| model | 向量模型 | varchar(45) |  |
| vector | 文本向量 | vector |  |
| team\_id | 租户ID | guid |  |
| dataset\_id | 知识库ID | guid |  |
| collection\_id | 文档ID | guid |  |

|  |  |  |  |

Team\_id，是用户ID，比如一个用户注销了，就可以直接删除这个租户的所有数据。

Dataset\_id，是知识库的ID，比如用户删除了一个知识库，就可以直接删除

Collection\_id，是文档ID，比如用户删除了一个文档，就可以直接根据它来删除向量

Model，所使用的向量模型，

对于片段的全文检索，也需要提供相应的字段。
